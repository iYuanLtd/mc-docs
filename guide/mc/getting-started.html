# 快速开始

提示

本游戏依赖环境Java，若您是第一次游玩此游戏，请下载Java安装
Java8 64位open in new window Java16 64位open in new window

# 下载启动器

  • 这里为您推荐 Plain Craft Launcher (又称 PCL) 启动器
    本教程将使用 PCL启动器 演示
    点击这里 下载启动器

提示

若您不喜欢PCL启动器 为您推荐其他两款优秀启动器
HMCL 启动器：MCBBSopen in new window
BakaXL 启动器：MCBBSopen in new window OfficalWebsiteopen in new window

  • 在电脑中新建一个文件夹 随意命名

  • 将下载的启动器放在此文件夹中

  • 双击启动器文件启动

# 下载游戏本体

提示

若您是正版玩家且使用官方正版启动器下载过游戏，PCL不会在此文件夹中新建 .minecraft 文件夹，而是会在您的AppData中的 .minecraft 里安装游戏

  • 下载游戏本体文件 这里下载1.17版本演示

  • 安装 Optifine

  • 选择第一个即可

  • 开始安装

# 启动游戏

  • 选择刚刚下载的版本

  • 选择 正版登录 或 离线登录 输入用户名
  • 启动游戏
